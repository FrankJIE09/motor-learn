行星齿轮

行星减速器 (Planetary Gear)

太阳轮、行星轮、内齿圈同轴布置，功率密度高。多级串联可达到高减速比。广泛应用于机器人关节、工业传动。

i = 1 + Zring / Zsun   η = 90–97%
减速比 3:1–100:1 级数 1–3 级 背隙 3–10 arcmin
谐波减速器

Harmonic Drive 应变波齿轮

柔轮 (Flexspline) 通过波发生器产生弹性变形，与刚轮 (Circular Spline) 啮合实现零背隙传动。单级可达超大减速比。

i = ZCS / (ZCS − ZFS)   K ≈ 103–104 Nm/rad
减速比 30–160 效率 70–85% 背隙 0 (零)
准直驱 QDD

Quasi-Direct Drive 准直驱

低减速比 (6:1–15:1) 行星减速 + 大转矩电机。反向驱动性好，力控带宽高，适用于动态性能要求高的足式机器人。

Tout = Tmotor · i · η   (i = 6–15)
减速比 6–15:1 效率 85–95% 代表 MIT Cheetah, H1
直驱 DD

Direct Drive 直接驱动

电机转子直接连接负载，无中间传动环节。i=1:1，效率 ≈ 100%，完全无背隙。但需要电机自身输出极大转矩，转矩密度较低。

Tout = Tmotor   ωout = ωmotor   η ≈ 100%
减速比 1:1 效率 ≈ 100% 背隙 0
差速器 / 蜗轮

蜗轮蜗杆 & 差速传动

蜗杆带动蜗轮实现垂直轴传动和大减速比。具有自锁特性（取决于导程角）。差速器用于车辆轮间转矩分配。

减速比 5–100:1 效率 50–85% 自锁 可能
误差放大

减速比 → 误差放大效应

高减速比会线性放大电机侧的转矩脉动、摩擦力矩、Kt 波动和传感器噪声。i=121 时，0.01 Nm 的电机误差放大为 1.21 Nm 的输出误差。

∆Tout = i × (∆Tcogging + ∆Tfriction + ∆TKt + ∆Tnoise)
i≤25 5–10% 误差 i≥80 30–50% 误差 Kt 3–5% 漂移

工程案例分析

减速比选型对力控精度的影响 (i=10 vs i=121)

⚡ 低减速方案 (i=10)

电机转矩 0.1 Nm → 输出 1.0 Nm
Kt 漂移 3% → 输出误差 3%
齿槽转矩 0.005 Nm → 输出 0.05 Nm
摩擦 5–10% 额定转矩
总误差 5–10% ← 可接受

🔴 高减速方案 (i=121)

电机转矩 0.01 Nm → 输出 1.21 Nm
Kt 漂移 3% → 输出误差 3% (绝对值放大121倍)
齿槽转矩 0.005 Nm → 输出 0.605 Nm
摩擦 10–20% 额定转矩
总误差 30–50% ← 需力传感器
