关节电机

SPMSM 无框电机

机器人关节广泛采用 SPMSM 无框电机（无外壳、无轴承），直接嵌入关节壳体。转矩密度 1.5–3.0 Nm/kg，Kt 温漂 −0.12%/°C (NdFeB 磁钢)。

Te = 3/2 · (P/2) · λPM · iq = Kt · iq
Kt 3–5% 波动 温漂 −6.6% (25→80°C) 转矩脉动 < 1%
位置传感

编码器 & 位置采样

电机端编码器 10–40 kHz 采样位置 θm，关节端编码器 1–5 kHz 采样 θj。分辨率需达 214–219 线/转以上。

∆θ = θm/i − θj   (弹性变形角)
电机编码器 10–40 kHz 关节编码器 1–5 kHz 分辨率 214–219 线/转
力矩传感

弹性变形力矩测量

通过测量弹性体两端的相对角位移 ∆θ 推算关节力矩。刚度为 Kstiffness，力矩分辨率由 ∆θmin 决定。

Tjoint = Kstiffness × ∆θ   ∆θmin = Tresolution / Kstiffness
Kstiffness 103–104 Nm/rad 分辨率 0.0011° (K=5000) 精度 5–10%
FOC 控制

磁场定向控制 + 电流采样

Clarke/Park 变换将三相电流解耦为 d-q 轴。iq 直接控制电磁转矩，电流环带宽 > 10 kHz。SPMSM 采用 id=0 策略。

Texpected = Kt · iq  →  Tout = Kt · iq · i · η
电流环 > 10 kHz PWM 40–80 kHz iq 转矩电流
双环融合

电流 + 力矩双环控制

电流环 Tcurrent = Kt · iq · i 提供高频低延迟响应；力矩环 Tdual = Kstiffness · ∆θ 提供精确直流校准。两者加权融合。

Tfinal = α · Tcurrent + (1 − α) · Tdual
α 高 响应快，噪声低 α 低 精度高，延迟大 H1 5–10% 精度
DOB 观测

干扰观测器 (DOB / ADRC)

基于模型估算外部干扰和未建模动态：力矩 = 电磁转矩 − 惯量加速 − 摩擦 − 已知干扰。DOB 可补偿 15–30% 的干扰量。

T̂dist = Kt · iq · i − J · ˙ω − T̂friction
DOB 15–30% 补偿 低减速 10–20% 提升 摩擦 10–20% Trated

三大机器人关节架构对比

基于《电机与传动系统底层原理》第7章

H1 — 准直驱关节

SPMSM 无框电机 + QDD 行星减速 (i≈10)
电机编码器 (214–219 线) + 关节编码器
弹性体 ∆θ 力矩传感
电流 + 力矩双环融合
力控精度 5–10%，反向驱动性好
WBC 带宽 > 500 Hz

Optimus — 高减速关节

SPMSM + 多级传动 (i≈100)
高分辨率编码器 (< 0.01° 分辨率)
关节力矩传感器 (直接测量)
电流环 + 力矩传感器闭环
力控精度 < 1%，高刚度
Kstiffness > 104 Nm/rad

Walker — 谐波减速关节

SPMSM + 谐波减速器 (i≈50–160)
电机编码器 + 关节编码器
关节力矩传感器 + 电流混合
电流 + 力矩传感器融合
力控精度 < 1%，零背隙
WBC 支持，力传感器要求高

电磁转矩 Te = 3/2 · (P/2) · λPM · iq
输出转矩 Tout = Kt(T) · iq · i · η − Tfriction − Tripple
关节力矩 Tjoint = Kstiffness · ∆θ
双环融合 Tfinal = α · Tcurrent + (1−α) · Tdual

反射惯量 Jreflected = Jload / i²

🎯 选型建议

高动态足式 → QDD 方案 (H1 路线)，反向驱动好
高精度操作 → 谐波/高减速 + 力矩传感器 (Optimus)
双足平衡 → 谐波减速 + 混合力控 (Walker)
低减速优势 高带宽 WBC，可电流环直接力控
高减速代价 需力矩传感器，摩擦/齿槽放大
